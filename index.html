AngularJS

Writing websites instantly becomes more fun and less tedious when using Anuglar JS. I love creating modular code using this framework. Bootstrap

Wow! Clean! Beautiful! Amazing! I love hearing this when people see the work that can be done with Bootstrap. Cascading Style Sheets can be used in many ways to describe how elements on a web page should look and feel. Bootstrap has done an amazing job keeping what we believe to be beautiful, consistent! Don't get me wrong, being different is great, and I love writing core CSS but using this framework has saved me so much time.

GitHub

I can no longer imagine software without source control, or i simply do not want to. GitHub has revolutionized open source software, making it accessible and easy to use, create and update code from all across the world. Gathering and organizing friends to work on a project is lightning fast and easy with GitHub, overall I love how far this ecosystem has came.
